# Assets Directory

Store your project assets here:

## /logos
- Brand logos (White Hat Coffee logo)
- Icon variants

## /images
- Product images (coffee drinks)
- Homepage hero images
- Shop/location images
- About section images

## /icons
- Small icons for UI
- Navigation icons
- Feature icons

## Usage in React

To use images in your components:

```jsx
// For images in public/assets/images/
<img src="/assets/images/coffee-drink.png" alt="Coffee" />

// For the logo
<img src="/assets/logos/white-hat-logo.png" alt="White Hat Coffee" />
```

**Note:** Files in the `public/` folder are served as static assets and can be referenced directly with `/`
